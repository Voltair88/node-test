
# Blog

A simple blog with mongoDB and login/signup.


## Tech Stack

**Client:** Ejs, Lodash

**Server:** Node, Express,


## Run Locally

Clone the project

```bash
  gh repo clone Voltair88/node-test
```

Go to the project directory

```bash
  cd node-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
